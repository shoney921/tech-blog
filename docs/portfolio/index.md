# 포트폴리오

다양한 비즈니스 도메인에서 성장한 **백엔드 개발자 이상헌**입니다.  
통신·금융·항공·제조·화학·안전 등 여러 분야의 프로젝트를 거치며, 요구사항 분석·설계·개발을 수행해 왔습니다. 최근에는 AI·LLM 기반 검색, RAG, 에이전트 설계 및 그래프 기반 데이터 분석 업무를 담당하고 있습니다.

---

## 프로필 요약

- **역할**: 백엔드 개발, 분석/설계/개발
- **소속**: LG CNS (2018.01 ~ 현재)
- **특징**: 고객 니즈를 정확히 파악하고, 도메인에 맞는 백엔드·데이터 솔루션 설계 및 구현. MSA, 이벤트 기반 아키텍처, AI/LLM 파이프라인 경험 보유.

---

## 주요 기술

| 구분 | 기술 |
|------|------|
| **Back-end** | Java, Spring Boot, DevOn Boot, BXM Framework, .NET Framework, Python, FastAPI |
| **DB** | Oracle, Maria DB, PostgreSQL, MS SQL, Neo4j, ChromaDB, FAISS |
| **AI·검색** | LangChain, LangGraph, RAG, 벡터 검색, Elasticsearch, 임베딩·그래프 기반 검색 |
| **기타** | REST API, MQ, Kafka, CQRS, MSA, AWS(EC2, KMS), Git, SVN, JUnit5, Streamlit |

---

## 회사 경력

**LG CNS** 정규직 (2018.01 ~ 현재)

백엔드 개발 인력이 필요한 프로젝트에 투입되어 다양한 고객사와 협업해 왔습니다.  
플랫폼엔지니어링팀에서 근무하며, 사내 POC부터 대형 고객사 프로젝트까지 분석·설계·개발을 수행하고 있습니다.

### 2026.01 ~ 2026.03 — Knowledge Graph 기반 CT 성적서 검색 및 위험성 예측 (내부 POC)

- **역할**: 분석/설계/개발
- **요약**: CT 성적서 도메인을 그래프로 모델링하고, 위험성 분석·예측 및 AI 의미 검색을 구현한 POC.
- **주요 업무**
  - 물질·용기·시험 조건·시험 결과 등 도메인 엔티티·관계 정의, RDF/LPG 비교 검토 후 **Neo4j 기반 그래프 설계**
  - 기존 성적서 데이터를 그래프로 변환, **그래프 경로·패턴 분석**으로 미시험 조합의 잠재 위험성 예측
  - 사용자 **자연어 질의 → 그래프 탐색 → 과거 위험 사례 분석** 및 자연어 설명을 제공하는 AI 검색 기능 구현

### 2025.11 ~ 2025.12 — 멀티 에이전트 기반 안전환경 관리 자동화 (내부 POC)

- **역할**: 분석/설계/개발
- **요약**: Super Agent와 다수 Sub Agent 구조로 안전환경 목표·실행·모니터링을 자동화하는 POC.
- **주요 업무**
  - **멀티 에이전트 아키텍처 설계**: KPI 기반 목표 설정, 계획 수립, Sub Agent 오케스트레이션
  - 외부 환경 변화 감지에 따른 **목표·실행 계획 동적 수정**, 에이전트 판단에 따른 Sub Agent 전략 변경
  - 랜덤 오류 시나리오로 **재시도·대응 흐름 검증**, 연간 목표 → 주 단위 KPI 기반 실행·반복 모니터링 구조로 개선

- **역할**: 분석/설계/개발
- **요약**: Advanced RAG 기반 JSA(작업안전분석) 위험성 평가 자동화.

  - 클러스터 기반 검색·메타데이터 필터링·컨텍스트 확장 등 **Advanced RAG 전략** 비교 적용, **FAISS·LangChain** 검색 파이프라인 구축

### 2025.03 ~ 2025.04 — LG화학 HAZOP 위험성 평가 자동 생성 AI 프로토타입

- **역할**: 분석/설계/개발
- **요약**: 도면 해석부터 위험원 도출·산출물 생성까지 LangGraph 기반 워크플로우로 구현한 프로토타입.
- **주요 업무**
  - **LangGraph** 기반 도면 해석 → 위험원 도출 → 산출물 생성의 **에이전트 워크플로우** 설계, 단계 간 상태 전이 정의
  - **OCR**로 도면 내 설비 정보 자동 추출, 설비 흐름 기반 공정 관계·위험 지점 분석
  - **3-Depth Advanced RAG**: 설비 유형 → 공정 조건 → 과거 사고 사례 순 컨텍스트 확장, **ChromaDB** 벡터 검색으로 위험원·대응 방안 도출
  - **Streamlit** 기반 UI로 위험성 평가 결과·분석 흐름 시각화
