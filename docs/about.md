# 소개

안녕하세요, **Shoney**입니다.

개발하면서 겪은 삽질, 새로 배운 것들, 그리고 아직 정리가 안 된 생각들을 여기에 기록하고 있습니다. 완성된 글보다는 과정 중에 남기는 메모에 가깝습니다.

## 저에 대해

다양한 비즈니스 도메인에서 성장한 백엔드 개발자입니다. 통신·금융·항공·제조 등 여러 분야의 프로젝트를 거치며, 요구사항을 정확히 파악하고 그에 맞춘 백엔드 개발을 해왔습니다. 최근에는 AI·LLM 기반 검색, RAG, 에이전트 설계 같은 작업도 하고 있습니다.

## 이 블로그에 대해

처음엔 노션에 정리하다가, 어차피 쓸 거 공개해두면 나중에 나한테도 편하겠다 싶어서 블로그로 옮겼습니다. 주로 다루는 주제는:

- **AI / LLM** — 모델 활용, 프롬프트 엔지니어링, 에이전트 개발
- **웹 개발** — 프론트엔드, 백엔드 이것저것
- **DevOps** — 배포 자동화, 클라우드 인프라

정해진 주기 같은 건 없고, 쓸 게 있으면 씁니다.

## 기술 스택

이 블로그는 [VitePress](https://vitepress.dev)로 만들었고, [Cloudflare Pages](https://pages.cloudflare.com)에서 호스팅합니다. 마크다운으로 글을 쓰면 빌드 후 자동 배포됩니다.

**업무에서 사용한 기술**

- **Back-end**: Java, Spring Boot, .NET Framework, Python, FastAPI
- **DB**: Oracle, MariaDB, PostgreSQL, MS SQL, Neo4j, ChromaDB, FAISS
- **기타**: REST API, MQ, Kafka, MSA, CQRS, AWS(EC2, KMS), Git, JUnit5, LangChain, LangGraph
